# AI News Daily

Ежедневная подборка новостей про искусственный интеллект.

## Запуск локально

```bash
npm install
npm run dev
```

## Деплой на Vercel

1. Пуши код в GitHub
2. Открой vercel.com
3. Import Git Repository
4. Deploy

## Переменные окружения

Создай `.env.local`:
```
TAVILY_API_KEY=твой_api_ключ
```

Или используй бесплатный поиск без ключа.
